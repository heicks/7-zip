7-Zip 插件

7-Zip 创建可以管理压缩档案。而且让所有的压缩档案在使用的时候就像浏览普通文件夹一样轻松。

用户绝大多数的操作都可以通过在 7-Zip 文件管理器(或 Windows 资源管理器)中点击右键来完成相应的操作。

使用 7-Zip

打开压缩档案

您可以通过两种方法来打开压缩档案：

  • 如果 7-Zip 默认关联了该压缩文件类型，您可以双击该文件来打开，或者右键点击要打开的压缩档案，选择 打开 命令。
  • 您可以右键点击压缩档案，指向 7-Zip 菜单，然后点击 打开 命令。
释放压缩档案

您可以通过两种方法来释放压缩档案中的文件：

  • 右键点击压缩档案文件，指向 7-Zip 菜单，然后点击 释放文件... 或 当前目录释放 命令。
  • 如果是释放指定的文件，您首先要 使用 7-Zip 打开压缩档案，然后选定要释放的文件或文件夹，最后执行 复制到... 命令。

另请见 解压缩对话框。

测试压缩档案

右键点击压缩档案文件，指向 7-Zip 菜单，然后点击 测试当前档案 命令。

创建及更新压缩档案

右键点击要压缩的文件和文件夹，指向 7-Zip 菜单，然后点击 添加到压缩档案... 命令，最后进行设置。详细请见 添加到压缩档案对话框。
