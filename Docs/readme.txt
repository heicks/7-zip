7-Zip 9.20
----------

7-Zip 适用于 Windows 95/98/ME/NT/2000/2003/XP/Vista/7。

7-Zip 版权所有 (C) 1999-2011 Igor Pavlov。

软件特征：

  - 使用了 LZMA 与 LZMA2 算法的 7z 格式 拥有极高的压缩比
  - 支持格式：
     - 压缩 / 解压缩：7z、XZ、BZIP2、GZIP、TAR、ZIP
     - 仅解压缩：ARJ、CAB、CHM、CPIO、CramFS、DEB、DMG、FAT、HFS、ISO、LZH、LZMA、
                 MBR、MSI、NSIS、NTFS、RAR、RPM、SquashFS、UDF、VHD、WIM、XAR、Z
  - 更快的压缩和解压缩速度
  - 7z 格式支持创建自释放压缩包
  - 为 7z 与 ZIP 提供更完善的 AES-256 加密算法
  - Windows 资源管理器集成
  - 强大的的文件管理器
  - 更给力的命令行版本
  - 共 79 种界面语言


7-Zip 是一款发布于 GNU LGPL 协议下的免费软件（除 unRar 代码及 AES 代码）
您可以查看 License.txt 文档了解详细的版权信息。


  7-Zip 当前的版本中包含以下文件：

  7zFM.exe      - 7-Zip 文件管理器
  7-zip.dll     - 7-Zip 插件
  7zg.exe       - 7-Zip 7-Zip GUI 插件模块
  7z.exe        - 7-Zip 命令行版本
  7z.dll        - 7z 运行库
  7z.sfx        - 自释放模块（Windows 版本）
  7zCon.sfx     - 自释放模块（命令行版本）

  License.txt   - 版权许可信息
  readme.txt    - 自述文档（当前文件）
  History.txt   - 7-Zip 历史
  7-zip.chm     - 用户手册（已编译的帮助文件）

  Lang\en.txt   - 英语语言文件
  Lang\*.txt    - 其它本地化语言文件
